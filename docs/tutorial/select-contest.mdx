# コンテストを選択

設定されたコンテストは、ダッシュボードのデータ表示の対象になったり、Editorの問題選択タブに問題が追加されます。

## 設定ダイアログを開く　

ホームのボタンか、メニューバーのボタンから設定ダイアログがひらけます。
<img
  src={require("./assets/select-contest/1.png").default}
  alt="login button"
  width="40%"
/>

## コンテストを選択する
これから問題を解くコンテストを選択します。  
リストには、開催予定のコンテストと、最近終了したコンテストがいくつか表示されています。  
リストにないコンテストを設定する場合は、AtCoderのURLに含まれるContestNameを入力します。
https&#58;//atcoder.jp/contests/**abc127**の場合
「abc127」と入力します。
<img
  src={require("./assets/select-contest/2.png").default}
  alt="login button"
  width="40%"
/>

## コンテストを設定できました！！ 🎉
